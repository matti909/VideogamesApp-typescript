# VideogamesApp-typescript

Descripción breve de tu aplicación.

## Tecnologías Utilizadas

- Frontend: React con TypeScript
- Backend: Node.js con Express.js
- Base de Datos: Postgresql con Typeorm

## Requisitos Previos

- Node.js (https://nodejs.org/)
- Gestor de paquetes npm (se instala junto con Node.js)
- Docker
- Docker-compose

## Instalación

1. Clona este repositorio
2. Navega hasta api2/docker
3. Completa las variables de entorno (archivo .env)
4. Instala dependencias con npm -i dentro de la capeta api2/ y luego en client/
